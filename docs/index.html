Task Bridge API

Mobil uygulama, web arayüzü ve otomasyon ajanları için tek backend. Epik oluşturma, workflow aşamaları, yorumlar, worker kuyruğu ve kütüphane bağlantıları bu API üzerinden yönetilir.

Base URL: http://localhost:3000 Format: JSON Auth: X-Api-Key

Varsayılan port

3000 — TASK_BRIDGE_PORT ile değişir

API anahtarı

BACKEND_API_KEY — her istekte header

Public endpoint

GET /health ve GET /connect.json auth gerektirmez

Kimlik doğrulama

Korunan tüm route'lar X-Api-Key header'ı bekler. Değer sunucudaki BACKEND_API_KEY ile eşleşmelidir. Yanlış veya eksik anahtar 401 Unauthorized döner.

ÖRNEK Herhangi bir korunan endpoint 
curl -s http://localhost:3000/projects \
  -H "X-Api-Key: dev-key"

Mobil uygulama GET /connect.json ile tünel bilgisini ve API anahtarını alır.

Hata formatı

Çoğu hata düz JSON nesnesi olarak döner.

{ "error": "Task not found" }
KodAnlam
400Geçersiz body veya iş kuralı ihlali
401API anahtarı eksik veya hatalı
404Kaynak bulunamadı
409Çakışma (ör. task zaten claim edilmiş)
503/connect.json — ngrok tüneli yok

Epics & Tasks

Epikler (parentId: null) workflow köküdür. Alt görevler epik veya başka bir göreve bağlanır. Detay yanıtında isEpic, subtasks, comments ve stage bulunur.

POST /epics Epik oluştur (workflow otomatik spawn)
AlanAçıklama
projectIdHedef proje
title / textEn az biri zorunlu; text ilk satır başlık olur
descriptionİsteğe bağlı uzun metin
 
curl -X POST http://localhost:3000/epics \
  -H "X-Api-Key: dev-key" \
  -H "Content-Type: application/json" \
  -d '{"projectId":"task-bridge","title":"Yeni özellik","description":"Detay..."}'
Yanıt 201
{
  "id": 42,
  "title": "Yeni özellik",
  "createdAt": "2026-05-25T10:00:00.000Z",
  "projectId": "task-bridge",
  "projectName": "Task Bridge",
  "parentId": null,
  "stageId": "stage-1",
  "assignee": "dev"
}

Worker

Otomasyon ajanları için kuyruk ve toplu claim.

GET /worker/pending Bekleyen görevler

Query: projectId, role, claimedBy (isteğe bağlı)

GET /worker/pending?projectId=task-bridge&role=dev
POST /worker/claim-next Sıradaki uygun görevi claim et 
{
  "claimedBy": "cursor-agent",
  "role": "dev",
  "projectId": "task-bridge"
}

404 uygun görev yok. Başarıda inbox item + comments döner.
